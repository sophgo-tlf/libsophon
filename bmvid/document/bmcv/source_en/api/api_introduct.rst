BMCV API
===============================
Briefly explain which part of the hardware implements the BMCV API

**The following interfaces BM1684X have not yet implemented:**

*  bmcv_image_canny
*  bmcv_image_dct
*  bmcv_image_draw_lines
*  bmcv_fft
*  bmcv_image_lkpyramid
*  bmcv_image_morph
*  bmcv_image_sobel
